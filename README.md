## Project setup
```
npm install
```

### Compiles and hot-reloads for development

```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Testing
```
# Run all tests
npm run test

# Run unit tests only
npm run test:unit

# Run e2e tests
npm run test:e2e

# Edit e2e tests
npm run dev:e2e
```

### Lints and fixes files
```
npm run lint
```
